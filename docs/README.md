## 🔧 技术栈概览

### 后端技术
- **框架**: FastAPI (Python 3.9+)
- **数据库**: SQLite 3.x
- **ORM**: SQLAlchemy 2.x
- **异步**: asyncio + Celery
- **浏览器**: DrissionPage + Selenium

### 前端技术
- **基础**: HTML5 + CSS3 + JavaScript (ES2022)
- **UI框架**: Bootstrap 5.3
- **图表**: Chart.js 4.x
- **图标**: Font Awesome 6.x

### 第三方集成
- **AI服务**: OpenAI GPT-4, Anthropic Claude, Google Gemini
- **检测服务**: 朱雀检测API
- **浏览器**: Fingerprint Chromium
